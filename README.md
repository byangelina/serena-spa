# Serena Spa

Aplicación web desarrollada con React + Vite para el spa Serena Spa,
un espacio de bienestar femenino en Santiago.

## Tecnologías

- React
- Vite
- LocalStorage (para el CRUD de reservas)

## Instalación

npm install

## Ejecución en desarrollo

npm run dev

## Build de producción

npm run build
